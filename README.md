## Preguntas
**¿Cuál es la diferencia entre autenticación y autorizacion?**
*R//* La autenticación verifica quién eres (iniciar sesión con email/password) y autorización decide qué puedes hacer una vez autenticado (verificar permisos).

**¿Cuál es la función del token JWT en la guía?**
*R//* El JWT prueba la identidad (autenticación) y da acceso a recursos protegidos (autorización básica).
